# synergy

A new Flutter project.

## Getting Started


1. Clone repository
2. flutter clean
3. flutter pub get
4. move terminal focus to domain package i.e. cd domain/ <br />
   i. flutter clean <br />
   ii. flutter pub get <br />
5. move terminal focus to data package i.e. cd data/ <br />
   i. flutter clean <br />
   ii. flutter pub get <br />
   ii. flutter pub run build_runner build <br />
6. run project
