## Getting Started


1. Clone repository
2. flutter clean
3. flutter pub get
4. move terminal focus to domain package i.e. cd domain/
   i. flutter clean
   ii. flutter pub get
5. move terminal focus to domain package i.e. cd data/ 
   i. flutter clean
   ii. flutter pub get
   ii. flutter pub run build_runner build
6. run project
